## springboot物流配送管理系统

> [点我下载源码或在线试用](https://www.notmaker.com/detail/17f11f5085754489a0ef70c5f1022bb4/ghb20250812) 

> 支持定制、讲解、修改、远程调试

### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Navicat`

### 三、功能描述
系统分为（身份）：管理员。

**管理员功能：**
1. 登录
2. 商品管理
3. 来往单位
4. 员工管理
5. 仓库管理
6. 销售开票
7. 销售记录
8. 申请配送
9. 销售记录
10. 车辆资料
11. 驾驶员资料
12. 入库分析
13. 出库分析
